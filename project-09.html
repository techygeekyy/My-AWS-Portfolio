Project 09

Portfolio Details

Multi OS-Platform EC2 Web Hosting
(Linux + Windows Servers)

AWS Cloud Platform

May 2024 Hybrid OS Cloud (Linux & Windows) Cross-Platform Web Hosting Operations

Project Overview

This project demonstrates the deployment and management of multi-platform web servers on AWS EC2, using both Linux and Windows instances to host web applications. The objective was to gain hands-on experience with different operating systems, web servers, and cloud networking concepts while simulating a real-world hosting environment accessible over the internet.

🎯 Objectives

  • Launch and configure Linux and Windows EC2 instances
  • Host web applications using Apache on Linux and IIS / NGINX on Windows
  • Configure security groups to allow controlled internet access
  • Understand cross-platform server management in AWS
  • Validate public access and server communication

🌍 Environment Details

  • ☁️ Cloud Provider: AWS
  • ☁️Region: ap-south-1 (Mumbai).
  • ☁️ Availability Zones: ap-south-1a, ap-south-1b

🧱 Architecture Components

  • 🏗️ EC2 Linux Instance:

    • 🔹Amazon Linux
    🔹Apache Web Server installed and configured

  • 🏗️ EC2 Windows Instance:

    • 🔹Windows Server
    🔹NGINX installed and configured

  • 🏗️ Security Group:

    • 🔹HTTP (80) / HTTPS (443) allowed from internet
    🔹SSH (22) and RDP (3389) restricted for administration

  • 🏗️ Public IP / Elastic IP:

    • 🔹Used to access web servers over the internet

🔁 Traffic Flow Explanation

  • 🔹 Users access the web application using a browser
  • 🔹 Requests reach the EC2 instance via the public IP
  • 🔹 Apache (Linux) and NGINX (Windows) process requests
  • 🔹 Web pages are served back to users over HTTP/HTTPS
    ☑️ Each server operates independently, simulating real-world heterogeneous infrastructure.

🔐 Security & Best Practices Implemented

  • 🛡️ Only required ports opened in security groups
  • 🛡️ SSH and RDP access restricted to trusted IPs
  • 🛡️ Separate instances for different operating systems
  • 🛡️ Principle of least privilege followed for access rules
  • 🛡️ Designed following real production standards

Through this project, I gained practical experience in deploying and managing both Linux and Windows EC2 instances in AWS. I learned how to configure web servers on different operating systems, manage instance access using SSH and RDP, and expose applications securely to the internet using security groups. The project helped me understand cross-platform server administration, cloud networking fundamentals, and how mixed OS environments are commonly used in real-world cloud infrastructures.
