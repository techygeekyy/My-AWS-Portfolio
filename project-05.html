Web Application with Amazon RDS in Private Subnet
(Secure 2-Tier Architecture)

Innovative Cloud Platform

January 1, 2026 Cloud Computing Simulated Enterprise Project

Project Overview

This project demonstrates the deployment of a secure two-tier web application architecture on AWS, where the web application runs on EC2 instances in a public subnet and the database (Amazon RDS) is hosted in a private subnet. The objective was to design a production-ready setup that follows best practices for network isolation, security, and data protection, ensuring that the database is never directly accessible from the internet.
